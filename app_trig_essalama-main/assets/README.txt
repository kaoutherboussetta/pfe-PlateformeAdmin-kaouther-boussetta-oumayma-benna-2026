Pour que la vidéo et le logo s'affichent dans l'app :

1. Vidéo de fond (intro) :
   - Placez le fichier "intro.mp4" dans le dossier : assets/videos/

2. Logo :
   - Placez le fichier "logo_trig_essalama.png" dans le dossier : assets/images/

Sans ces fichiers, l'app affichera un fond noir et une icône de remplacement pour le logo.
